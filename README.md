# js.lib

## Features

- base64
  - encode
  - decode
- list
  - filterReduce
  - foldl
  - foldr
  - take
  - takeWhile
- heap
  - number
    - MinHeap
- node
  - fs
    - readFileSync
    - writeFileSync


## Usage

```bash
npm install @f0c1s/js.lib
```

```javascript
const {readFileSync, writeFileSync} = require("@f0c1s/js.lib/node/fs");

```
